The Last Of Us Wiki

Wiki completa e interativa dedicada ao universo de The Last of Us. Projeto desenvolvido com HTML, CSS que reúne informações detalhadas sobre personagens, enredo, armas, locais e curiosidades dos jogos The Last of Us Part I e Part II.

Recursos principais:

  • ✅ Navegação intuitiva por categorias
  • ✅ Galeria de imagens oficial
  • ✅ Banco de dados completo de itens e armas
  • ✅ Design imersivo
  • ✅ Efeitos visuais inspirados no jogo

Tech Stack: HTML5 | CSS3

Stock Control

Sistema completo de gerenciamento de estoque desktop desenvolvido em **Java 25** com banco **MySQL**. Aplicação Maven com interface gráfica para controle total de produtos, entradas/saídas e relatórios.

🚀 Funcionalidades:

  • 📦 Cadastro/edição de produtos
  • 📊 Controle de estoque (entrada/saída)
  • 🔍 Pesquisa e filtros avançados
  • 📈 Relatórios de movimentação
  • 💾 Banco MySQL com script SQL incluso

Tech Stack:

  • ☕ Java 25 (Maven)
  • 🗄️ MySQL 8+
  • 🖥️ Swing/JavaFX (interface desktop)
  • 📁 JDBC (conexão banco)
